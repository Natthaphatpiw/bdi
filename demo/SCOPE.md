# ขอบเขตระบบ รู้สิทธิ์ รู้สุข (ฉบับทีมใช้ตอบกรรมการ)

> Source of truth เชิงโปรแกรมอยู่ที่ `demo/scope.ts` — การลดหรือขยาย scope ต้องแก้ไฟล์นั้นก่อนเสมอ ไฟล์นี้คือฉบับอ่าน

## ระบบทำอะไรได้ (in-scope)

### 1. กลุ่มอาการที่รองรับ (12 คลัสเตอร์)
ไข้/หวัด/เจ็บคอ · ปวดหัว/ไมเกรน · เวียนหัว/บ้านหมุน · ปวดท้อง/ท้องเสีย/อาหารเป็นพิษ · ปวดฟัน/เหงือก · ปวดหลัง/กล้ามเนื้อ/ข้อ · บาดแผลเล็กน้อย/ไฟไหม้เบา · ผื่น/แพ้ · ตาแดง/ตาพร่า · อาการโรคเรื้อรังพบบ่อย (เบาหวาน/ความดัน) · ทางเดินหายใจ (ไอเรื้อรัง หอบเบา) · ระบบประสาทไม่วิกฤต (ชา เหน็บ มือสั่นเล็กน้อย)

แต่ละคลัสเตอร์ map ไประดับบริการแบบ deterministic: ร้านยาคุณภาพ (กลุ่ม 32 อาการ) / คลินิกชุมชนอบอุ่น–รพ.สต. / OPD โรงพยาบาล — โดย **เด็กอายุต่ำกว่า 15 ปี ไม่แนะนำ self-care/ร้านยาเป็นทางหลัก** เสมอ

### 2. Red flags (ตรวจแบบ deterministic ก่อน LLM เสมอ — เข้าเส้นทางฉุกเฉินใน turn เดียวกัน)
เจ็บ/แน่นหน้าอก (รวมร้าวแขน-กราม) · หายใจลำบาก/หอบรุนแรง · หมดสติ/เรียกไม่รู้ตัว · อ่อนแรง-ชาครึ่งซีก/ปากเบี้ยว/พูดไม่ชัดเฉียบพลัน · ปวดหัวรุนแรงเฉียบพลันแบบไม่เคยเป็น · ชัก · อาเจียน/ถ่ายเป็นเลือดมาก · ตั้งครรภ์ + ปวดท้อง/เลือดออก · ซึมลง เหงื่อแตก ตัวเย็น · แพ้รุนแรง (หน้าบวม หายใจติด) · เด็กไข้สูง+ซึม/ชัก · ภาวะวิกฤตสุขภาพจิต (พ่วงสายด่วน 1323)

### 3. สิทธิ์ที่รองรับ
- บัตรทอง — ทั้งสิทธิ์ใน กทม. และ **สิทธิ์ต่างจังหวัด + 30 บาทรักษาทุกที่**
- ประกันสังคม ม.33/39 — รวมทันตกรรม 900 บาท/ปี และการเปลี่ยน รพ. ตามสิทธิ์
- ข้าราชการ/เบิกได้ — ระดับข้อมูลทั่วไป
- **"ไม่รู้สิทธิ์"** — ระบบซักสถานะอาชีพเพื่ออนุมาน แล้วให้ผู้ใช้ยืนยันเสมอ (ไม่เดาเงียบ ๆ)
- สิทธิเสริม: เบี้ยยังชีพผู้สูงอายุ, UCEP, ตรวจคัดกรองตามช่วงวัย

### 4. ผู้ป่วยที่เล่าแทนได้
ตัวเอง · ผู้ใหญ่ในครอบครัว (พ่อ/แม่/คู่สมรส) · เด็ก <15 ปี (เส้นทางอนุรักษ์นิยม: route คลินิก/รพ. และ red flag เด็ก → ฉุกเฉิน)

### 5. พื้นที่
ข้อมูลสถานพยาบาลครอบคลุม **กรุงเทพฯ + ปริมณฑล** (นนทบุรี ปทุมธานี สมุทรปราการ สมุทรสาคร นครปฐม) — ต่างจังหวัดไกล: ตอบเรื่อง **สิทธิ์** ได้เต็มรูปแบบ แต่ระบบจะแจ้งตรง ๆ ว่า facility data รุ่นนี้ครอบคลุม กทม. แล้วชี้ช่องทาง 1330

## ระบบไม่ทำอะไร (out-of-scope + พฤติกรรม fallback)

| คำขอ | พฤติกรรมระบบ |
|---|---|
| ขอวินิจฉัย/ยืนยันโรค | ปฏิเสธอย่างสุภาพ + เสนอประเมินเบื้องต้น/เตรียมข้อมูลไปพบแพทย์ |
| ขอขนาดยาเฉพาะบุคคล | ไม่ให้ dosing — แนะปรึกษาเภสัชกร + route ร้านยาคุณภาพ |
| ตั้งครรภ์เชิงลึก | red flag (ปวดท้อง/เลือดออก) → ฉุกเฉิน; อื่น ๆ แนะฝากครรภ์/สายด่วน |
| สุขภาพจิตวิกฤต | เส้นทางฉุกเฉิน + สายด่วนสุขภาพจิต 1323 |
| นอกเรื่องสุขภาพ | บอกขอบเขต + ชวนกลับเรื่องสุขภาพ/สิทธิ์ |
| ภาษาอื่น | ตอบไทย + ประโยคอังกฤษแจ้งว่าบริการรุ่นนี้เป็นภาษาไทย |
| คดี/ร้องเรียนแพทย์ | ให้ช่องทาง สบส. 1426 / แพทยสภา อย่างเป็นกลาง ไม่ตัดสินฝ่ายใด |
| เลขบัตรประชาชน | ไม่รับ ไม่เก็บ ไม่ทวนซ้ำ — แจ้งว่าไม่จำเป็นต้องใช้ |
| ขอคุยกับคนจริง | การ์ดช่องทางมนุษย์: 1330 (สปสช.) / 1506 (ปกส.) |

## หลักการที่ระบบรับประกัน (invariants)
1. ทุกคำตอบจบด้วย actionable next step อย่างน้อย 1 อย่าง
2. ทุกการอ้างสิทธิ์มี citation (ชื่อประกาศ + วันที่มีผล)
3. Red flag → เส้นทางฉุกเฉินภายใน turn เดียวกัน ไม่มีข้อยกเว้น
4. ไม่มีชื่อโรคเป็นข้อสรุป / ไม่มี dosing เฉพาะบุคคล / ไม่มี auto-call
5. เคสเดโม่วิ่ง pipeline เดียวกับทุกเคส — ไม่มี code path พิเศษ

## ประโยคตอบกรรมการเมื่อออกนอกขอบเขต
"ระบบรุ่นนี้ประกาศขอบเขตชัดเจนที่ 12 กลุ่มอาการ + 4 สิทธิ์หลัก ครอบคลุมกรุงเทพฯ และปริมณฑล — นอกขอบเขตนั้นระบบจะบอกตรง ๆ และส่งต่อช่องทางที่ถูกต้อง (1330/1669/1323) เสมอ ไม่เดาคำตอบ"
